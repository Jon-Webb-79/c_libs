This macro will allocate memory for an array and assign that memory to the array pointer variable in a struct. The macro will also assign the number of allocated indices to allocated_length and assign active_length a size of 0. This function requires the input of a struct instantiated with the VECTOR_STRUCT method.

/param vec A vector instantiated with the VECTOR_STRUCT macro. /length The length of the array as a size_t variable

// Initialize a dynamically allocated Integer vector with 0 indices
Int vec;
INIT_VECTOR(vec, 20)

This macro will produce a struct that is specifically designed to track all attributes necessary for the maintenance of a dynamically, or statically allocated array. The macro has an attribute to a pointer for the array, allocated_length, active_length, and mem_type.

/param type The data type that will be assigned to the array pointer /param dtype The name to be given to the typedef for the struct

// code to create a struct of type IntVector
