int append_array ( Array *  array,
void *  elements,
size_t  count 
)

This function allows a user to append an existing or blank array container with scalars or arrays of any data type except strings.

Parameters
arrayA pointer to the memory location where an array exists
elementsA scalar or an array of variables. The variables can be of any data type, so as long as they are of the same data type as the array being appended.
countThe size of the elements data type in number of indices they will occupy
Returns
integer A 0 or 1 indicating success of the function.

The following demonstrates several ways that the append_array function can be uses. In these instances, the function will be used for an array of integers

// Instantiate array
size_t indices = 20;
char dtype[4] = "int";
char name[5] = "Array";
Array arr_test = init_array(dtype, indices, name);
// Build array from scalar values
for (int = 0; i < 3; i++) {
append_array(&arr_test, &i, 1);
}
// Print the array values
for (int i = 0; i < 3; i ++) {
printf("%d\n", ((int *) arr_test.array)[i]);
}
